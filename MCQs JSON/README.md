# DHAMCQ Exam Simulator

A professional MCQ exam simulator platform with 10 test sets, 151 questions each, 3-hour timer, and comprehensive score tracking.

## Features

- 📚 10 complete test sets with 151 questions each
- ⏱️ 3-hour timer with auto-submit
- 📱 Fully responsive design (works on desktop, tablet, and mobile)
- 🎯 Question navigation with 151 buttons
- 📊 Real-time progress tracking
- ✅ Answer review with correct/incorrect/skipped filtering
- 🔄 Reset test option
- 📈 Score calculation with percentage
- ⏰ Completion time tracking
- 🎨 Modern, clean UI

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript (Vanilla)
- Font Awesome icons (via CSS pseudo-elements)
